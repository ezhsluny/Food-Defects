# Food-Defects

## Файловая структура датасета
```bash
apples_dataset
├── train
│   ├── crit_def
│   ├── mid_def
│   ├── no_def
│   └── small_def
└── val
    ├── crit_def
    ├── mid_def
    ├── no_def
    └── small_def
```

- crit_def - критический дефект
- mid_def - значительный дефект
- no_def - нет дефекта
- small_def - незначительный дефект
